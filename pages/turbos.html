Turbo kits

Kits de alto rendimiento para vehículos turbos, incrementan la potencia de tu motor hasta un 40%

Turbo kit desarrollado para VW Vento motor 2.0TSI, potencia máxima 430hp. Para su intalación se requiere stg2

Bmw 335i

Turbo kit desarrollado para Bmw 335i motor N54, potencia máxima 700hp. Para su instalación se requiere stg2 y methanol

Toyota Gt86

Turbo kit desarrollado a medida para toyota Gt86, potencia máxima 350hp. Para su instalación se requiere enfriadores de aceite

Bmw E36

Turbo kit desarrollado a medida para Bmw e36 motor M50/52, potencia máxima 450hp. Para su intalación no se requiere ningun agregado

Honda Civic Si

Kit de compresora desarrollada para Honda Civic SI, potencia máxima 510hp. Para su cololación no se requiere de ningun agregado

Golf GTI

Turbo kit desarrollado para Volkswagen Golf GTi MK7/7.5, potencia máxima 600hp. Para su instalación se requiere stg2 y kit methanol
